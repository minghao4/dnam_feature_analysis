====
Workflow for BISULFITE VCF
====

User inputs:
- Minimum coverage

Input file:
Biscuit generated VCF

1. Remove metadata from VCF:
    - Cut every line before "#CHRM..."
    - Remove every variant with LowQual
    - Remove every variant under user-indicated min. coverage

2. Process VCF file into following columns:
    - Chromosome
    - Position
    - Reference Allele
    - Alternate Allele
    - 2 Nucleotide Context
    - 5 Nucleotide Context
    - Format
    - Genotype
